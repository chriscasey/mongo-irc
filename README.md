mongo-irc
=========

This is a demo/experimental irc server that uses MongoDB for meta data, message bus and logging.

To run on port 6667:

  * `ant run`

To run on port 6668:

  * `ant run2`

To run on port 80:
  * `ant run80`


Extra Features:

  * to get logs: `/LOGS #fun`

See:
http://www.mongodb.org/
